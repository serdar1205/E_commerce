# E-commerce app

This project has written in clean architecture.

## App contains

  - Flutter Clean architecture
  - Bloc state management
  - DI get_it
  - Routing: without any package used Navigator
  - Local storage
  - dio package used to fetch remote data

## Features
 - authentication
 - favorites
 - basket
 - history of ordered products
 - category
 - contact us
 - filtering products
 - pick image
